# Claude Code Digest

Telegram-канал **@claudecode_digest** — практический дайджест по Claude Code для разработчиков.

## Миссия

Помогать разработчикам внедрять Claude Code в ежедневную работу. Не пересказ документации, а реальный опыт, находки и паттерны.

## Автор

Полгода ежедневной работы с Claude Code. Построил несколько production-систем где AI — ключевая часть.

## Проекты-кейсы

### BlueFreedom — AI Trading Platform
- **Repo:** ~/Workspace/bluefreedom (мета), ~/Workspace/crypto/* (сервисы)
- **Суть:** Автоматическая торговля на Bybit. Claude Sonnet анализирует рынок каждые 30 мин и генерирует торговые сигналы. Multi-source архитектура (trend following + AI + партнёры).
- **Стек:** Python, FastAPI, Redis pub/sub, Docker, Telegram Bot, WebSocket
- **Контент-углы:**
  - AI-агент как источник торговых сигналов
  - CLAUDE.md на 700+ строк для сложной системы
  - CI/CD и автодеплой через GitHub Actions
  - Multi-service архитектура управляемая через Claude Code

### Umka — Semantic Video Search
- **Repo:** ~/Workspace/umka (уточнить)
- **Суть:** Задай вопрос — получи ответ из видео. Семантический поиск по видеоконтенту.
- **Контент-углы:**
  - AI для обработки мультимедиа
  - Семантический поиск и embeddings
  - Как Claude Code ускорил разработку

## Контент-стратегия

### Рубрики

1. **Фичи и обновления** — разбор новых возможностей Claude Code с практическими примерами
2. **Workflow** — как организовать работу с CC (CLAUDE.md, hooks, MCP, skills)
3. **Кейсы** — реальные проекты (BlueFreedom, Umka, другие)
4. **Инструменты** — полезные MCP серверы, хуки, шаблоны
5. **Tips** — короткие практические советки

### Формат постов

- Короткие (300-500 слов), конкретные
- Скриншоты и фрагменты кода
- Один пост = одна идея
- Теги: #фича #workflow #кейс #инструмент #tip

### Частота

2-3 поста в неделю. Стабильность важнее объёма.

## Монетизация (долгосрочно)

1. **Консалтинг** — внедрение Claude Code в команды (канал = лидген)
2. **Open-source инструменты** — hooks, MCP серверы, шаблоны (GitHub Sponsors)
3. **Воркшопы** — "Claude Code для тимлидов" (когда аудитория 3k+)

## Skills

При создании поста Claude Code должен:

1. Писать на русском языке
2. Использовать разговорный но профессиональный тон
3. Добавлять конкретные примеры кода или команд
4. Держать длину 300-500 слов (если не указано иное)
5. Один пост = одна чёткая мысль
6. Включать 1-3 хештега в конце
7. НЕ использовать emoji в тексте (только в заголовках если уместно)
8. Формат: Telegram-совместимый markdown
