# Пост 1: Вступительный

**Что это за канал и зачем он тебе**

Claude Code — это CLI-агент от Anthropic, который пишет, рефакторит и деплоит код прямо из терминала. Не автокомплит, не чат-бот. Полноценный агент, который читает твой проект, понимает архитектуру и вносит изменения сам.

Я работаю с ним каждый день уже полгода. За это время:

— Построил trading platform из 7 микросервисов, где AI-агент анализирует рынок и генерирует торговые сигналы каждые 30 минут
— Сделал систему семантического поиска по видео — задаёшь вопрос, получаешь ответ с таймкодом
— Научился кодить с телефона, делегируя всё Claude Code

Этот канал — про практику. Что реально работает, какие паттерны ускоряют разработку в разы, и где подводные камни.

**Что здесь будет:**

— Разбор новых фич Claude Code с примерами
— Рабочие workflow: CLAUDE.md, хуки, MCP-серверы, субагенты
— Кейсы из реальных проектов
— Короткие tips которые экономят время

**Чего здесь не будет:**

— Пересказа документации
— Хайпа ради хайпа
— Постов ради постов

2-3 раза в неделю. Коротко и по делу.

Продукту год. Темпы роста — безумные. Русскоязычного контента почти нет. Этот канал закрывает эту дыру.

Если работаешь с кодом — подписывайся. Будет полезно.
